# fine

## 简介

网络空间资产测绘工具，目前支持fofa、hunter、quake和零零信安，以及一些其他功能。

## 更新内容

**2023.07.10 v0.0.13** 

1. 添加下载列表功能；
2. 优化导出文件功能；
3. 修复已知翻页问题;
4. 去除IP显示功能。
